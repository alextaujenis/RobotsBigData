Robots + Big DataDOCUMENTATION

an open-source framework for connecting sensors to the internet

Overview

  • Sensor Libraries interact with electronic components and the real world, are written in C++, and run on any Arduino.

  • Server Libraries help move and process data, are written in CoffeeScript, and run on Node.js and the Raspberry Pi.

  • Client Libraries help display and interact with data, are written in CoffeeScript, and run on any modern web or mobile browser.
