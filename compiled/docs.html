An open-source framework for connecting sensors to the internet.

Install Guide

API Docs Overview

  • Arduino Sensor Libraries are written in C++, run in real-time with no delay on any Arduino, and interact with electronic components in the real world.

  • Raspberry Pi Server Modules are written in CoffeeScript, run in real-time without blocking on Node.js, and help move and process data.

  • Web & Mobile Client Plugins are written in CoffeeScript, run in real-time without blocking on most modern web browsers, and help display and interact with data.

  • Other Included Packages provide details on the specific implementation of external open-source projects within this framework.
