API Docs Overview

  • The Arduino Sensor Libraries are written in C++, run in real-time with no delay on any Arduino, and interact with electronic components in the real world.

  • The Raspberry Pi Server Modules are written in CoffeeScript, run in real-time without blocking on Node.js, and help move and process data.

  • The Web & Mobile Client Plugins are written in CoffeeScript, run in real-time without blocking on most modern web browsers, and help display and interact with data.

  • The Other Included Packages provides details on the specific implementation of external open-source projects within this framework.
