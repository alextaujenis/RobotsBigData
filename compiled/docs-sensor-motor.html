Motor API DOCUMENTATION

A real-time library for controlling many motors.

Download  See Example  Public Methods

Overview

A simple, non-blocking, real-time Arduino library for controlling many motors. It provides basic control along with more advanced features such as precision easing of speed over time.

Download

This library is part of a larger package that you can download to access all of the documented sensors, or you can use this library outside of the package with only one dependency.

  • Download Full Sensor Package with this library included
  • Download Motor Library with Timer dependency
  • View Source on GitHub with MIT license

EXAMPLE SKETCH

motor.update()

You can use this library without calling update() and most methods will function correctly.

You must continuously call .update() inside of loop() in order to use these specific time-based methods:

  • .timedOn()
  • .rampUp()
  • .rampDown()
  • .ramp()
  • .rampPercent()

EXAMPLE SKETCH

motor.on()

Start the motor at full speed.

EXAMPLE SKETCH

motor.off()

Stop the motor.

EXAMPLE SKETCH

motor.timedOn(timeout)

Start the motor at full speed and turn it off after the specified time in milliseconds.

EXAMPLE SKETCH

motor.isOn()

Returns true if the motor is running at any speed greater than 0%.

EXAMPLE SKETCH

motor.isFullOn()

Returns true only if the motor is running at 100% speed.

EXAMPLE SKETCH

motor.isOff()

Returns true only if the motor is at 0% speed.

EXAMPLE SKETCH

motor.getPwm()

Returns an integer from 0 - 255 for the current motor speed.

EXAMPLE SKETCH

motor.getPwmPercent()

Returns an integer from 0 - 100 for the current motor speed percentage.

EXAMPLE SKETCH

motor.setPwm(value)

Pass in an integer from 0 - 255 to control the speed of the motor.

EXAMPLE SKETCH

motor.setPwmPercent(value)

Pass in an integer from 0 - 100 to control the speed percentage of the motor. This is essentially the same as setPwm() but with a smaller input scale.

EXAMPLE SKETCH

motor.rampUp(timeout)

Pass in a timeout in milliseconds for how long it will take to ramp from the current speed to full speed with a linear transition.

EXAMPLE SKETCH

motor.rampDown(timeout)

Pass in a timeout in milliseconds for how long it will take to ramp from the current speed to full stop with a linear transition.

EXAMPLE SKETCH

motor.ramp(value, timeout)

Pass in a value from 0 - 255 to control the target speed of the motor, and a timeout in milliseconds for how long it will take to ramp to that target speed from the current speed with a linear transition.

EXAMPLE SKETCH

motor.rampPercent(value, timeout)

Pass in a value from 0 - 100 to control the target speed of the motor, and a timeout in milliseconds for how long it will take to ramp to that target speed from the current speed with a linear transition. This is essentially the same as ramp() but with a smaller input scale.
