Light API DOCUMENTATION

Control many lights.v1.0.0

Download  See Example  Public Methods

Overview

Control many lights and blink without delay using commands such as on(), off(), blink(), and fade(). Lights must be connected to PWM enabled Arduino pins to work with this library.

Download

This library is part of a larger package that you can download to access all of the documented sensors, or you can use this library by itself with one dependency.

  • Download Light Library v1.0.0  by itself with the Timer dependency, or
  • Download All Sensor Libraries with this library and dependency included, or
  • View Source on GitHub with MIT license

Public Methods

You can use this library without calling update() and most methods will function correctly. You must continuously call update() inside of loop() in order to use theREAL-TIME methods tagged below.

  • .on()

light.update()PROCESS REAL-TIME

Keep the light moving in real-time. This must be called continuously from within loop() in order to blink() or fade() the light.

EXAMPLE SKETCH

light.blink(on_time, off_time, times1)REAL-TIME

Pass in unsigned longs for the time values in milliseconds. Pass in an integer for the last parameter for the number of times the light should blink. The last parameter is optional1 and can be left out to make the light repeat the blink cycle forever.

  • the light will turn on for the on_time
  • then turn off for the off_time
  • and will repeat this cycle for the number of times you provided1
EXAMPLE SKETCH

light.fade(up_time, on_time, down_time, off_time, times1)REAL-TIME

Pass in unsigned longs for the time values in milliseconds. Pass in an integer for the last parameter for the number of times the light should fade. The last parameter is optional1 and can be left out to make the light repeat the fade cycle forever.

  • the light will perform a linear fade up according to the up_time
  • stay on for the on_time
  • linear fade down for the down_time
  • then stay off for the off_time
  • and will repeat this cycle for the number of times your provided1
