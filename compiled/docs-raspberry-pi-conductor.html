Conductor API DOCUMENTATION

Load json config files into memory.v1.0.0

Download  See Example  Public Interface

Overview

This non-blocking module accepts a list of JSON config files, reads them from disk, and makes the config data available in memory. This is helpful for first starting an application and getting credentials or address information, then connecting to authenticated or external services such as a database or remote camera.

Download

This module is part of a larger package that you can download to access all of the documented server modules, or you can use this module by itself.

  • Install All Raspberry Pi Server Modules with this module included
  • OR, Download this module by itself from theCommand Line with npm
  • View Source on GitHub with MIT license
