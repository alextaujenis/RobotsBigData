Light API DOCUMENTATION

A real-time library for controlling many lights.

Download  See Example  Public Methods

Download

This library is part of a larger package that you can download to access all of the documented sensors, or you can use this library by itself with one dependency.

Public Methods

You can use this library without calling update() and most methods will function correctly. You must continuously call update() inside of loop() in order to use the REAL-TIME methods tagged below.

  • constructor(pin)
  • .update()PROCESS REAL-TIME
  • .on()
  • .off()
  • .isOn()
  • .isOff()
  • .setPwm(value)
  • .setPwmPercent(value)
  • .getPwm()
  • .getPwmPercent()
  • .blink(on_time, off_time, times)REAL-TIME
  • .pulse(up_time, on_time, down_time, off_time, times)REAL-TIME

Light constructor(pin)

Create a new light and pass in the Arduino pin number.

light.update()PROCESS REAL-TIME

Keep the light moving in real-time. This must be called continuously from within loop() in order to blink() or pulse() the light.

light.setPwm(value)

Pass in an integer between 0 - 255 to set the brightness of the light.

EXAMPLE SKETCH

light.setPwmPercent(value)

Pass in an integer between 0 - 100 to set the brightness of the light.

EXAMPLE SKETCH

light.getPwm()

Returns an integer of the current light brightness from 0 - 255.

EXAMPLE SKETCH

light.getPwmPercent()

Returns an integer of the current light percentage brightness from 0 - 100.

EXAMPLE SKETCH

light.blink(on_time, off_time, times)REAL-TIME

Pass in integers for the time values in milliseconds.

  • the light will turn on for the on_time
  • then turn off for the off_time
  • and will repeat this cycle for the number of times you provided
EXAMPLE SKETCH

light.pulse(up_time, on_time, down_time, off_time, times)REAL-TIME

Pass in integers for the time values in milliseconds.

  • the light will perform a linear fade up according to the up_time
  • stay on for the on_time
  • linear fade down for the down_time
  • then stay off for the off_time
  • and will repeat this cycle for the number of times your provided
