API Docs Overview

  • The Sensor libraries interact with electronic components and the real world, are written in C++, and run on any Arduino.

  • The Server libraries help move and process data, are written in CoffeeScript, and run on Node.js and the Raspberry Pi.

  • The Client libraries help display and interact with data, are written in CoffeeScript, and run on any modern web or mobile browser.
