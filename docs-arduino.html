Arduino LibrariesAPI DOCUMENTATION

Real-time event based C++ libraries written for the Arduino.

How to Install

  • Components

  • ButtonRead and debounce buttons and switches.
  • LightControl many lights.
  • MotorControl many motors.
  • ServoControl many servos without delay or interrupts.

  • Sensors

  • Light SensorRead and calibrate photoresistors.
  • Water SensorMeasure and calibrate water level sensors.
  • Human SensorDetect human presence, touch, and pickup events.

  • Additional Libraries

  • TimerManage many timed events.
  • ThresholdSet and check numeric quantile scales.
  • CapacitanceMeasure change in capacitance.
  • Serial ManagerA simple interface for serial communication.
